History

Kotlin is an OSS statically typed programming language that targets the JVM, Android, JavaScript and Native. The project started in 2010 and was open source from very early on. The first official 1.0 release was in February 2016. Kotlin was invented to solve developer challenges of JetBrains. They needed a simpler tool than Java to work with their main product, called IntelliJ IDEA, totally written in it. The guys investigated the alternatives, stayed unsatisfied, and decided to invent their own language.
