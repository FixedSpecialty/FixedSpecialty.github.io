Hello, Kotlin.

Welcome to Kotlin!

The first steps to writing your very own program are to set up the tools necessary for compiling and executing your files. If you have not done this yet, setup instructions can be found under Resources. Once you've done this, you are ready to start coding!

Let's start with the very basics.
Create a new Kotlin file, named HelloWorld.kt (Kotlin files use the extension .kt, and open the file in the editor of your choice. Writing the following code, compiling, and executing your program will make it say hello! 
                
fun main(){
    println("Hello, world! It's Kotlin speaking.")
}
Kotlin programs are usually made up of Functions. In our HelloWorld program, we only have one function-- the main function. Functions are discussed in more detail in a later tutorial.

If you haven't set up your program to execute on your computer just yet, you can execute the same code in the browser! In the box below, try editing the program to say something new!

//This is a comment! In Kotlin, a comment can begin with two forward slashes. //A comment has no effect on your programs execution, so you can type whatever you want. fun main(){ println("What should I say...?") //Try typing something new inbetween the quotes! }
