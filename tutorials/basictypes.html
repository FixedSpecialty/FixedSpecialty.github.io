Basic Types

Back

In Kotlin, everything is an object. This means that you can call member functions and properties on all of you variables!

The basic types of Kotlin include:

  • Numbers
  • Characters
  • Booleans
  • Arrays
  • Strings

Numbers

val one = 1 // Int
val threeBillion = 3000000000 // Long
val oneLong = 1L // Long
val oneByte: Byte = 1
val pi = 3.14 // Double
val e = 2.7182818284 // Double
val eFloat = 2.7182818284f // Float, actual value is 2.7182817

Something to note is that functions can only return the type declared in the parameters. There is no implicit conversion. The code below can only print doubles. fun main()
{
fun printDouble(d: Double) { print(d) }
}

Characters

Character literals go in single string quotes like 'c'. Characters cannot be treated directly as numbers but we can explicitly convert a character to an Int number.

Booleans

Boolean has two values similar to Java: True or False
Built-in operations include
  • ||
  • &&
  • !

Arrays

To create an array use the library function arrayOf() and pass in values. val array = arrayOf(1,2,3) //[1,2,3] val x: IntArray = intArrayOf(1, 2, 3) //[1,2,3] val x: ByteArray = byteArrayOf(1, 2, 3) //[1,2,3] val x: ShortArray = shortArrayOf(1, 2, 3) //[1,2,3]

Strings

Strings are represented by the type String and are immutable. Elements in a string can be indexed into as well. String str = "abcd"
for (c in str)
{
println(c)
}

For more information on Basic Types go to KotlinLang.org
