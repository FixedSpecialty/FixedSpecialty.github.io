Classes in Kotlin are declared like so,

class Example {/*class content*/}

A class has a primary constructor that can be written into the class with return type written after the ':' character

class Example constructor(exibitA: String) {/*class content*/}

Since the primary constructor can not have any code inside of it, put all of your code inside and initializer block using the keyword init. class Example(exibitA: String) {
init {
println("First initializer block that prints ${exibitA}")
}

init {
println("Second initializer block that prints ${exibitA.length}")
}
} This will print First initializer block that prints exibitA
Second initializer block that prints 6 For more information on Classes go to KotlinLang.org
